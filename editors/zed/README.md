## Installation

### Zed Marketplace
The extension has not yet been published in the marketplace. See PR [here](https://github.com/zed-industries/extensions/pull/4122).

### Manual Installation
- Clone this repository
    ```bash
    git clone git@github.com:nurulhudaapon/ziex.git
    ```

- Open the Extensions panel in Zed (CTRL/CMD + SHIFT + P)
- Select `Install Dev Extension`
- Navigate to the `editors/zed` directory in the cloned repository
